## Структура мастер-класса

Каждый материал хранится в `content/<slug>/`.

Минимальные артефакты:

- `brief.md` — утверждённая идея и фокус.
- `scenario.md` — сценарий записи.
- `recording-brief.md` — ТЗ для человека, который записывает экран.
- `voiceover-script.md` — финальная озвучка по реальному видео.
- `edit.py` — данные сегментов для `pipeline/pipeline/editor.py`.
- `final.mp4` — готовое видео.
- `article.md` — статья.
- `article-images/` — скриншоты для статьи.
- `metadata.yaml` — title, description, keywords, tool, level.

Тяжёлые raw/work файлы игнорируются через `.gitignore`.
