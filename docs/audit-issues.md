# Аудит проекта

Список проблем, найденных через skill `audit`.

## Текущий статус

Публичный шаблон инициализирован. Новые аудиты добавлять сюда короткими блоками:

```markdown
### YYYY-MM-DD — Аудит #N

Контекст: что проверяли.

#### Найдено
- A1. ...

#### Исправлено
- ...
```

## Категории

- **A** — критические: блокируют работу агента или пайплайна.
- **B** — высокие: заметно влияют на качество результата.
- **C** — средние: создают неудобства или риск ошибок.
- **D** — низкие: косметика, навигация, формулировки.
