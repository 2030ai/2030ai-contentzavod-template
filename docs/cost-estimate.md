# Оценка расходов

Ориентир для производства обучающих материалов. Фактические цены зависят от тарифов сервисов и региона.

## Базовые затраты

- Agent runtime: подписка/тариф вашей AI-среды.
- FFmpeg: бесплатно, локально.
- Edge TTS: бесплатно, подходит для черновиков.
- ElevenLabs: платный production TTS, если нужен естественный голос.
- Whisper verification: локальная модель `tiny`, требуется extra `verify`.
- Иллюстрации: опционально через OpenAI API или другой image provider.

## Человеческое время

- Выбор и утверждение идеи: 5-10 минут.
- Запись экрана по `recording-brief.md`: 10-20 минут.
- Финальный просмотр: 5-10 минут.

## Что влияет на стоимость

- Длина ролика и количество TTS-сегментов.
- Количество перегенераций TTS после Whisper-проверки.
- Наличие image generation для статей.
- Использование платных API вместо локальных/fallback-инструментов.
