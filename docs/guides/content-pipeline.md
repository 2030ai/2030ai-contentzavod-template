# Гайд: пайплайн генерации контента

Полный цикл: запрос пользователя -> готовый пакет (видео + статья + скриншоты + мета).

## Обзор цепочки

**Фаза 1 — Валидация идеи** (`pipeline-phase1-validation.md`):
Режим А (придумай): [Исследователь] Шаг 1 -> [Фокус-группа] Шаг 2 -> [Контролёр] Шаг 3 -> ПАУЗА
Режим Б (вот идея): [Фокус-группа] Шаг 2Б -> [Контролёр] Шаг 3Б -> ПАУЗА

**Фаза 2 — Производство** (`pipeline-phase2-production.md`):
[Редактор] Шаг 4 -> [Режиссёр] Шаг 5 -> ПАУЗА ->
[Контролёр] Шаг 6 -> [Монтажёр] Шаг 7 -> [Контролёр] Шаг 8 -> [Редактор] Шаг 9 -> [Контролёр] Шаг 10 -> [Очистка] Шаг 11

## Два режима входа

**Режим А — «Придумай»:** пользователь даёт тему/инструмент, агент генерирует идеи.
Проходит шаги 1 -> 2 -> 3.

**Режим Б — «Вот идея»:** пользователь даёт конкретную идею, агент валидирует и дорабатывает.
Шаг 1 пропускается, проходит шаги 2Б -> 3Б.
