# Навигация по документам

Короткая карта проекта. Открывать только релевантные файлы.

## Структура

| Папка | Назначение |
|---|---|
| `docs/` | Концепция, архитектура, гайды, история |
| `content/` | Создаваемые мастер-классы и их статусы |
| `pipeline/` | Python-код монтажа, TTS, извлечения кадров и тесты |
| `.agents/skills/` | Canonical skills `mk` и `audit` |

## Основные документы

- `docs/concept.md` — миссия, аудитория, уровни, формат, границы.
- `docs/architecture.md` — архитектура пайплайна и текущие ограничения.
- `docs/adr.md` — архитектурные решения.
- `docs/development-history.md` — журнал изменений публичного шаблона.
- `docs/roadmap.md` — дальнейшие улучшения шаблона.
- `docs/cost-estimate.md` — ориентир по стоимости производства.
- `docs/audit-issues.md` — результаты запусков `audit`.
