## Записи

### 2026-06-02 ADR-001: Видео-первичный пайплайн

#### Статус
Принято.

#### Контекст
Для обучающих материалов важно совпадение озвучки, экранных действий и результата. Если писать озвучку до записи, текст часто не ложится на реальное видео.

#### Решение
Сначала сценарий и ТЗ на запись, затем ручная запись экрана, анализ кадров, финальная озвучка по реальному видео, монтаж и только после этого статья.

#### Последствия
- Человек нужен для записи экрана и контрольных решений.
- Агент получает устойчивые артефакты: `scenario.md`, `recording-brief.md`, `voiceover-script.md`, `edit.py`, `article.md`.
- Пайплайн проще переносить между инструментами: браузер, IDE, документы, таблицы.

### 2026-06-02 ADR-002: ElevenLabs с Whisper-проверкой как production TTS

#### Статус
Принято.

#### Контекст
Нужна естественная русская озвучка. TTS иногда искажает слова или вставляет лишние слоги.

#### Решение
Production-путь: ElevenLabs multilingual voice. Каждый TTS-файл проверяется Whisper tiny; при расхождении генерация повторяется до 3 раз. Edge TTS остаётся fallback для разработки, Yandex SpeechKit — опциональная альтернатива.

#### Последствия
- Для production-видео нужен `ELEVENLABS_API_KEY` и выбранный `ELEVENLABS_VOICE_ID`.
- Для проверки нужен extra `verify`: `pip install -e ".[verify]"`.
- Правила произношения описаны в `docs/guides/voiceover-writing.md`.

Шаблон записи: `docs/templates/adr.md`.
