# Реестр опубликованного контента

Источник истины для материалов, которые уже опубликованы. Для всех материалов, включая черновики, см. `content/README.md`.

|---|---|---|---|---|
| _пока пусто_ |  |  |  |  |

## Статусы

- **Черновик** — материал собран, но ещё не опубликован.
- **Опубликован** — доступен на внешней площадке.
- **Требует обновления** — интерфейс/факты устарели или прошло больше 30 дней.

## Что проверять при обновлении

1. Не изменился ли интерфейс инструмента.
2. Работают ли промпты и шаги.
3. Актуальны ли скриншоты.
4. Обновлены ли `metadata.yaml` и дата в статье.
